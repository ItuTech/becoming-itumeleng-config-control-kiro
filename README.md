# Becoming Itumeleng — Kiro AI Config

This repo contains the personal AI team and steering rules powering the **Becoming Itumeleng** system, built inside [Kiro](https://kiro.dev).

The system is a collection of custom agents and tone guidelines that support me across content creation, personal brand, goal tracking, consistency, and confidence as I build in public.

---

## 🌟 The AI Team

| Agent | Role |
|---|---|
| **Becoming Itumeleng** | Lead coordinator. Knows the full system and routes to the right agent. |
| **Brand Guide** | Keeps my voice and messaging consistent across YouTube, LinkedIn, and Medium. |
| **Accountability Partner** | Tracks goals and checks in on progress. Kind but honest. |
| **Consistency Coach** | Builds daily habits and tracks streaks. Systems over goals. |
| **Hype Woman** | Celebrates wins, boosts confidence, and reminds me how far I have come. |
| **Style Curator** | Suggests outfits based on occasion, weather, mood, and what I actually own. |

---

## 🧠 Steering

The `steering/` folder contains rules that guide tone, writing style, and content standards across all agents.

`brand-and-tone.md` — the core identity and voice guidelines for the Becoming Itumeleng system.

---

## ☁️ How to use this

These configs work with [Kiro](https://kiro.dev). To use them:

1. Clone this repo into your `~/.kiro/` directory, or copy the `agents/` and `steering/` folders there.
2. Open Kiro and your agents will be available under the agent panel.
3. The steering file applies automatically to all interactions.
